Légende

  • Caméras de reconnaissance faciale : pour pouvoir traquer les citoyens.
  • Application mobile : possibilité de visualiser le score CSSE personnel, son propre classement et le classement global et un onglet pour pouvoir dénoncer d’autres personnes.
  • Dashboard : utilisé par le gouvernement pour visualiser toutes les informations sur les citoyens.
  • Base de données : pour le stockage des informations de la population.
  • API : outil pour coordonner le tout.

Dashboard général pour le gouvernement contenant toutes les informations utiles à la gestion du CSSE dans le pays comme par exemple:

  • Chiffres et graphiques de la consommation et production d’énergie du pays, des villes, de chaque personne, sur le jour, la semaine, le mois ou l’année.
  • Annuaire de la population contenant les informations de chaque individu comme le nom, prénom, adresse, mais surtout leurs informations concernant le CSSE comme leur localisation en temps réel, consommation et production d’énergie.
  • Gestion des dénonciations entre individus

Caméras de reconnaissance faciale

Elles seront installées partout dans le pays, permettant de traquer les citoyens pour savoir où ils sont et ce qu’ils font afin de croiser les informations avec leur position et savoir où ils ont marché et ce qu’ils ont produit. (Prototype non fonctionnel pour le moment)

Application mobile du gouvernement

Chaque citoyen devra l'avoir sur son téléphone. Il accèdera à son profil qui contiendra son score CSSE, son classement et le classement global. Les citoyens auront aussi un onglet pour pouvoir dénoncer d’autres personnes qui consommerait trop (leurs voisins, collègues, famille ...). L’application récupère aussi le podomètre du téléphone et les coordonnées GPS de la personne pour l’envoyer au gouvernement via l’API. Notre prototype fonctionnel est codé avec react js et ionic.

Base de données

Afin de récolter et stocker toutes les informations de la population et pouvoir les utiliser afin de réaliser des graphiques et statistiques. Prototype réalisé avec postgreSQL dans un conteneur Docker.
